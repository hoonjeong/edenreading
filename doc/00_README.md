## 🎯 빠른 이해를 위한 추천 순서

- **경영진/원장님**: `01` → `02` → `08`
- **기획자**: `01` → `02` → `03` → `07` → `08`
- **디자이너**: `01` → `02` → `07` → `04` + `05`
- **개발자**: `01` → `03` ~ `06` → `07` → `08`

## 📊 Mermaid 다이어그램 보는 방법

`.mermaid` 파일은 다음 방법 중 하나로 시각화할 수 있습니다:

1. **VS Code**: "Markdown Preview Mermaid Support" 또는 "Mermaid Preview" 확장 설치
2. **온라인**: https://mermaid.live 에 파일 내용 붙여넣기
3. **Notion / Obsidian**: ` ```mermaid ... ``` ` 블록으로 변환하여 붙여넣기

## 🔑 핵심 차별 포인트 요약

1. **학부모 인증코드 가입**: 무분별한 가입 차단, 교육원이 발급한 코드로만 가입
2. **자동 채점 + 리포트**: 정답 입력 후 답안만 입력하면 즉시 채점·리포트 자동 생성
3. **데이터 격리**: 학부모는 본인 자녀의 정보만 조회 가능 (행 단위 보안)
4. **멀티미디어 활동 기록**: 사진·영상·글을 한 화면에서 자유롭게 조합
