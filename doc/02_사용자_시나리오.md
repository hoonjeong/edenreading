# 사용자 시나리오

각 사용자가 실제로 시스템을 어떻게 사용할지 시간 순서로 기술합니다.
기획 검증과 화면 설계의 기준이 되는 핵심 시나리오입니다.

---

## [관리자 시나리오 1] 신규 학부모·학생 등록과 첫 활동 공유

**상황**: 김선생님이 새로 등록한 윤서(초3) 학생의 첫 수업을 마쳤다.

1. 김선생님, PC로 시스템에 로그인 → 대시보드 진입
2. **[학부모 관리]** 메뉴에서 "학부모 등록" 클릭
3. 학부모 정보 입력 (이름, 휴대폰, 이메일) → **6자리 인증코드 자동 발급**
4. 학부모에게 카톡으로 "이든톡 가입 안내 + 인증코드" 자동 발송
5. **[학생 관리]** 메뉴에서 "학생 등록" → 윤서 정보 입력 → 방금 등록한 학부모와 연동
6. 수업이 끝난 후 **[활동 기록]** → 윤서 선택 → "이번 주 활동" 업로드
   - 책 읽는 사진 3장 + 짧은 소감 글 + 토론 영상 1개
7. "저장 및 공유" 클릭 → 학부모에게 자동 알림 발송
8. 김선생님은 학부모가 잘 받았는지 알림 발송 결과 확인

**성공 기준**: 학부모 등록부터 첫 활동 공유까지 5분 이내, 학부모는 카톡으로 알림을 받고 즉시 확인 가능

## [관리자 시나리오 2] 분기별 시험 채점과 리포트 공유

**상황**: 2분기 국어 능력 평가일. 30명의 학생이 응시했다.

1. **[시험 관리]** 메뉴에서 "신규 시험 회차 생성"
2. 시험명: "2분기 국어 능력평가", 문항 수: 30개 입력
3. 각 문항의 **유형(어휘/독해/문법/추론)**, **정답**을 미리 입력
4. "응시 학생 배정" → 30명 일괄 선택
5. 학생별로 답안지를 보며 **답안만 빠르게 입력**
6. 입력 즉시 **자동 채점 완료** → 학생별 점수, 영역별 정답률 표시
7. **리포트 미리보기**: 회차별 변화 그래프, 영역별 강점/약점, 학년 평균 대비 위치
8. 모든 학생 응시 결과 입력 완료 → "전체 학부모에게 일괄 공유" 클릭
9. 학부모 30명에게 자동 알림 발송 + 학부모 페이지에 리포트 게시

**성공 기준**: 1명당 답안 입력 1분 이내, 채점·리포트 생성은 자동, 30명 처리 30분 이내

## [관리자 시나리오 3] 상담 요청 처리

**상황**: 정민이 어머니가 상담을 요청했다.

1. 대시보드에 "신규 상담 요청 1건" 알림 표시
2. **[상담 관리]** 메뉴 클릭 → 정민 어머니 요청 상세 확인
3. 어머니가 남긴 상담 희망일과 문의 내용 확인
4. 김선생님이 "상담 일정 확정" → 일시 입력 → 학부모에게 알림 발송
5. 상담일 당일 또는 직후, 상담 내용을 **상담 기록**에 저장
6. 향후 정민이 학생 카드를 열면 누적 상담 이력이 함께 표시됨

---

## [학부모 시나리오 1] 가입 후 첫 자녀 활동 확인

**상황**: 윤서 어머니가 교육원에서 받은 가입 안내 카톡을 받았다.

1. 카톡 메시지의 가입 링크 클릭 → 모바일 가입 화면 진입
2. 인증코드 6자리 입력 → 본인 정보(이름, 휴대폰) 확인
3. 비밀번호 설정 → 가입 완료
4. 자동 로그인 → **자녀(윤서) 카드가 표시된 홈 화면** 진입
5. "이번 주 활동" 카드 탭 → 윤서가 책 읽는 사진과 토론 영상 시청
6. "댓글" 또는 "좋아요"로 선생님께 반응 (선택 기능)
7. 다음 알림이 올 때까지 자유롭게 둘러보기

**성공 기준**: 가입에서 첫 활동 확인까지 3분 이내, 모바일에서 모든 미디어가 부드럽게 재생

---

## [학부모 시나리오 2] 시험 리포트 확인

**상황**: 윤서 어머니가 "분기 시험 결과가 도착했어요" 알림을 받았다.

1. 알림 탭 → 자동으로 시험 리포트 화면 이동
2. 윤서의 이번 회차 점수와 등급 확인
3. **영역별 그래프**: 어휘 ↑ / 독해 → / 문법 ↓ / 추론 ↑ 시각적 확인
4. **회차별 추이**: 1분기 → 2분기 점수 변화 라인 그래프 확인
5. 선생님 코멘트(선택) 함께 확인
6. 필요 시 "상담 신청" 버튼으로 상담 요청

---

## [학부모 시나리오 3] 자녀 독서 기록 회고

**상황**: 학기말, 윤서 어머니가 한 학기 동안 자녀가 읽은 책을 돌아보고 싶다.

1. 홈 화면에서 **[독서 기록]** 메뉴 진입
2. 기간 필터: "이번 학기" 선택
3. 윤서가 읽은 책 목록이 카드 형태로 표시 (책 표지 사진, 제목, 읽은 날짜)
4. 각 책 카드 탭 → 그날의 활동 사진과 선생님이 남긴 소감 확인
5. "전체 다운로드" 또는 "공유" 기능으로 가족과 추억 공유 (선택 기능)

---

## 시나리오에서 도출된 핵심 요구사항

| 요구사항 | 출처 시나리오 |
|---------|-------------|
| 학부모 등록 → 인증코드 자동 발급 → 카톡 발송 자동화 | 관리자1, 학부모1 |
| 한 화면에서 사진·영상·글 동시 업로드 | 관리자1 |
| 답안만 입력하면 자동 채점·리포트 생성 | 관리자2 |
| 영역별·회차별 시각화 그래프 | 관리자2, 학부모2 |
| 학생별 누적 상담 이력 조회 | 관리자3 |
| 모바일 우선 학부모 UI / PC 우선 관리자 UI | 학부모1, 관리자1 |
| 활동/시험 업로드 시 학부모 자동 알림 | 모든 시나리오 |
